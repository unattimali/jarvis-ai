# 🤖 Jarvis AI Assistant

A modern AI-powered desktop assistant built using **HTML, CSS, JavaScript, Python, Flask, and Ollama**. Jarvis provides a sleek futuristic interface, voice input, theme customization, real-time AI responses, and an expandable architecture for adding intelligent features.

---

## 🚀 Features Completed

### 💬 AI Chat

* Interactive chat interface
* Real-time conversation with Ollama (Llama 3.2)
* User and AI message bubbles
* Press **Enter** to send messages

### 🧠 AI Backend

* Python backend using Flask
* Connected to Ollama
* Generates AI responses locally
* REST API communication between frontend and backend

### 🎨 Modern UI

* Futuristic dashboard
* Responsive layout
* Sidebar navigation
* Top navigation bar
* Animated AI logo
* Live greeting message

### 🌗 Theme Customization

* Cyber Blue Theme
* Matrix Green Theme
* Neon Purple Theme
* Iron Red Theme
* Light Theme
* Theme settings panel
* Theme preference saved using Local Storage

### 🎤 Voice Input

* Speech Recognition using Web Speech API
* Voice-to-text message input
* Microphone button support

### ⏰ Live Features

* Real-time digital clock
* Dynamic greeting (Morning, Afternoon, Evening)

### 💾 Chat Features

* Chat history saved in Local Storage
* Quick suggestion buttons
* Scrollable conversation window

### ⚡ User Experience

* AI Thinking Animation
* Loading animation before AI replies
* Responsive design
* Smooth transitions and hover effects

### 🔗 Backend Integration

* Flask API
* JavaScript Fetch API
* JSON request/response handling
* Connected to local Ollama server

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Jarvis-AI-Assistant.git
```

### 2. Open Project

```bash
cd Jarvis-AI-Assistant
```

### 3. Install Dependencies

```bash
pip install flask
pip install flask-cors
pip install ollama
```

### 4. Install Ollama

Download and install Ollama from:

https://ollama.com/

### 5. Download Llama Model

```bash
ollama pull llama3.2
```

### 6. Start Ollama

```bash
ollama serve
```

### 7. Run Flask Backend

```bash
python app.py
```

### 8. Open Frontend

Open **ai.html** using VS Code Live Server.
